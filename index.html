A Few Accomplishments

Why am I a good fit for your organization? Have a look at few of my favorite projects I’ve been a part of during the course of my career.

Geo Tracker

Android, Java, SQL, Python

An android based application that lets you track your current location based on GPS or Network Provider and exports Geo Data to a CSV.

Simple Amazon DynamoDB

Android, Java, Socket Programming, SQL, DynamoDB

A simplified version of Amazon DynamoDB in Java, using 5 Android Virtual Devices acting as nodes. The design takes care of Partitioning, Data Replication, Node Failures and Replica Synchronization & Conflict Resolution using Object Versioning.

Software Defined Routing and DVRP Implementation

C, C++, UNIX Network Programming, GNU

Implemented a simplified version of a router in C++, which performs Control Plane and Data Plane functionalities and performs routing updates in a Distance Vector Routing fashion.

Distributed Hash Table

Android, Java, Socket Programming, SQL, Chord DHT

A simple DHT based on the Chord algorithm in Java, using 5 Android Virtual Devices acting as nodes. The model is an efficient Distributed Data Storage system that offers fast lookup times and is capable of handling ID space portioning/re-partitioning, Ring based routing and joining of new nodes to the networked system.

Distributed Message and File Sharing System

C, C++, UNIX Network Programming, GNU

Designed a shell-based Multi-Client Chat Application and File Sharing System in C++, under GNU environment. The application works on top of the CS Department servers at the University at Buffalo. This application implements both Client-Server and P2P model and provides functionalities like: Client login logout and active tracking, message storage and delivery guarantee and blocking client access.

Post-Quantum Primitives for Constrained Devices

Python, C, C++

Performed a comparative study of Lattice-based cryptographic algorithms and evaluated their performance by testing them on constrained device environment (Raspberry Pi B+). Programmed the following algorithms in Python and C: LP, U-LP, BLISS and NTRU.

Reliable Transport Protocol

C, C++, GNU

Implemented Reliable Data Transfer Protocols by programming them at Transport Layer in C++ to send and receive data reliably. The model implements (Go-Back-N, Alternating-Bit-Protocol and Selective Repeat), which transmit data at L4 using a simulated network environment. Did performance comparison of throughput of the three protocols using different window sizes and loss probabilities.

Social Media Indexer and Boolean Query Retrieval System

Java, Json, HTML, Solr, XML, Apache

Designed and developed a complete search-based solution in Java for parsing and indexing of a corpus of multilingual Tweets (English, German, French and Russian) through cross-lingual queries. The model implements Faceted Search, Content Tagging and provisions language detection at run time and index time.

Free wireless IP Telephony

Cisco IOS, GNS, VoIP

A model that is capable of offering free wireless telephony between two closely spaced university departments. The model is based on a call manger which is setup virtually using a network simulator on a server. For better voice reception, QoS was implemented on the model making it much more scalable.

Android Group Messenger with Total and FIFO ordering

Android, Java, SQLite, ISIS

A group messenger that guarantees in order message delivery by implementing the ISIS algorithm.
